# goadmin-soybean-sponge

## Project Description
This is a demp project for a web application, which using golang and vue.
The related web frameworks are:
| Framework Name| Note |
| --------    | --------     |
[Sponge](https://github.com/go-dev-frame/sponge) | The background tool, which can generate code, build on Gin/Gorm/...|
[Soybean](https://github.com/soybeanjs/soybean-admin)| The frontend tool, which can using Vue/React, such as Element-Plus/AntDesign/...|
| | |
