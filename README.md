# goadmin-soybean-sponge 

## Project Description [中文](./docs/README-CN.md)

This is a demp project for a web application, which using golang and vue.
The related web frameworks are:
| Framework Name| Note |
| --------    | --------     |
[Sponge](https://github.com/go-dev-frame/sponge) | The background framework, which can generate golang code, build on Gin/Gorm/...|
[Soybean](https://github.com/soybeanjs/soybean-admin)| The frontend framework, which can using Vue/React, such as Element-Plus/AntDesign/...|
| | |


## Prepare for development
- [vue tools](https://docs.soybeanjs.cn/guide/quick-start.html#environment-preparation)
- [golang tools](https://go-sponge.com/getting-started/install.html)



## How to run this project

### Commands
```shell
cd background
go mod tidy
start.sh
```
```shell
cd web
pnpm install
start.sh
```
